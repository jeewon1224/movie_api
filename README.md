# Movie API Project
Movie Api 프로젝트는 오픈 API를 활용하여 제작된 영화 사이트입니다.  Ajax로 영화 데이터를 수신하여 조회가 가능하게 제작하였습니다.</br>
영화 검색, 장르별 영화 보기, 신작 보기 등의 기능을 제공합니다.</br>

메인 페이지에서 영화 검색이 가능</br>
- Genre </br>
장르 별 영화 조회</br>
- Recent </br>
최신 개봉 영화 조회</br>
- Search Result </br>
검색된 영화 목록 조회</br>
